About This Site

This website showcases my work for the Computer Vision course. Each assignment submission includes:

  • Questions and problem statements specified
  • Images that are required in solutions to demonstrate principles
  • Answers, solutions, and explanations
  • Interactive voting system for peer review

For Assignment 0, students can vote on each problem to provide feedback. Login is required for voting. Login can be done using students' Ashoka IDs. All content is available otherwise, but voting is ONLY for approved IDs of students in the current Spring 2026 cohort.
